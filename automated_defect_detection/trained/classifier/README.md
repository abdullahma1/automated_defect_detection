This folder will contain trained classifier artifacts generated by `scripts/train_classifier.py`.

Artifacts produced:
- best.pt (PyTorch state dict)
- best.onnx (for OpenCV `cv2.dnn.readNetFromONNX`)
- best.torchscript.pt
- label_map.json
- metrics.json
- train_log.csv
- confusion_matrix.png
- splits.json

Train example:
  python scripts/train_classifier.py --data_root PATH/TO/DATASET --epochs 30 --batch_size 32 --img_size 224

Infer example:
  python scripts/infer_classifier.py --weights automated_defect_detection/trained/classifier/best.pt --image PATH/TO/IMAGE.jpg
